About

This project was created as the final project for the 2023-2024 CAS Computer Science class at Crossroads School for Arts & Sciences in Santa Monica, California. The dataset was taken from a dataset titled "California House Price" by Shibu Mohapatra in Kaggle. The project used a Python notebook to load the data set, explore and preprocess the data, and perform both a linear regression and a random forest model. These tasks were critical to analyze the big data and create accurate models that used the data to predict house prices. Using HTML, CSS, and Python, an effective website was created, allowing the user to estimate a the price of a specific property.
